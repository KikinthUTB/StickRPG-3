# Stick RPG 2 - Reference Overview

## Purpose
This folder contains **Accurate** documentation of *Stick RPG 2: Director's Cut*.
It serves as a factual roadmap and inspiration board for the development of *Stick RPG 3*.

## File Guide

### 01-MainStory.txt
*   Walkthrough of the **4D Object** Quest (Hypercube, Klein Bottle, Mobius Strip).
*   The 3 Endings (TV, Menu, NGP).
*   Positive vs Negative Karma paths.

### 02-Islands-and-Locations.txt
*   Breakdown of the 4 Islands (Slums, Residential, Business, Future).
*   Correct location of amenities like the Gym, University, and Casino.

### 03-Items-and-Prices.txt
*   Real item lists (Pecan Pie, Rad Bull, Crackbook Pro).
*   Weapons (Railgun, Lightsabers).
*   Vehicles (Hoverboard).

### 04-Jobs-and-Careers.txt
*   Complete profession list (Pimp, Hacker, Light Knight, Burger Flipper).
*   Salary and Stat requirements.

### 05-Characters-and-NPCs.txt
*   The cast of oddballs (Sad Keanu, Nigerian Prince, Prof Ansel).
*   Factions and interactions.

### 06-Gameplay-Mechanics.txt

*   Combat and Economy mechanics.

### 07-Minigames-and-Activities.txt

### 08-Audio-and-Atmosphere.txt

### 09-UI-and-User-Experience.txt

### 10-Art-Style-and-Aesthetics.txt
*   The "Paper Thin" 2.5D isometric look.
*   Visual humor and animations.

## Summary of Stick RPG 2's "Vibe"
Stick RPG 2 is not a serious life Simulator. It is a **Satire**.
*   **Humor:** Internet culture, edgy humor, absurdity.
*   **Freedom:** Be a cop or a drug dealer. Save the world or destroy it.
*   **Progression:** Simple numbers go up (Stats) -> Visual rewards (cool weapons/houses).

**Use this reference to ensure Stick RPG 3 captures the soul of the series!**
