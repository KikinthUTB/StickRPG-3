# Stick RPG 3

A modern recreation and continuation of the classic Stick RPG series.

## About

Stick RPG 3 is a life simulation game where you control a stick figure character in an open world environment. Build your stats, earn money, develop relationships, and make choices that shape your character's destiny.

## Features

- Open world exploration
- Character progression system
- Multiple jobs and career paths
- Social interactions and relationships
- Day/night cycle
- Various locations to discover
